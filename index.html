MHO API

MHO Movie DB - Movie Hub Offical Movie Database

MHO API is a RESTful web service to obtain movie information, this api is made using OMDB API. This service is free. The language used to create this API is Node JS.

Poster API

You can get the IMDB movie poster from here

This poster API is based on OMDB

Usage

gets movie info :

https://mho-movie-api.herokuapp.com/api/movie/[title of film]

gets movie poster :

https://mho-movie-api.herokuapp.com/api/poster/[imdb id for film]
